# 简单的C程序

这是一个简单的C程序，演示了基本的输入输出功能。

## 功能特性

- 获取用户的姓名、年龄和身高信息
- 根据年龄判断是否为成年人
- 计算建议的体重范围（基于BMI）
- 使用中文界面，用户友好

## 如何编译和运行

### 方法1：使用Makefile（推荐）

```bash
# 编译程序
make

# 运行程序
make run

# 清理生成的文件
make clean
```

### 方法2：直接使用gcc

```bash
# 编译
gcc -Wall -Wextra -std=c99 -o hello hello.c

# 运行
./hello
```

## 程序示例

运行程序后，您会看到类似以下的交互：

```
欢迎使用简单的C程序！
===================
请输入您的姓名: 张三
请输入您的年龄: 25
请输入您的身高(米): 1.75

个人信息摘要:
姓名: 张三
年龄: 25 岁
身高: 1.75 米
状态: 成年人
建议体重范围: 56.6 - 76.3 公斤

谢谢使用！
```
