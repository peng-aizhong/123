# 简单Java程序

这是一个简单的Java程序，演示了Java的基本功能。

## 文件说明

- `HelloWorld.java` - 主程序文件
- `run.sh` - 编译和运行脚本
- `README.md` - 说明文档

## 程序功能

这个程序演示了以下Java基本功能：
- 基本数据类型（int, String, double）
- 变量声明和赋值
- 基本运算（加法、乘法）
- 条件语句（if-else）
- 循环语句（for循环）
- 控制台输出

## 运行方法

### 方法1：使用脚本（推荐）
```bash
./run.sh
```

### 方法2：手动编译运行
```bash
# 编译
javac HelloWorld.java

# 运行
java HelloWorld
```

## 系统要求

- Java Development Kit (JDK) 8 或更高版本
- Linux/Unix 系统（用于运行脚本）

## 预期输出

程序运行后会显示：
- 欢迎信息
- 基本数据类型示例
- 简单计算示例
- 条件判断示例
- 循环计数示例
